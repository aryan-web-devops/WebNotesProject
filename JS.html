Introduction

JavaScript is a scripting language used to make webpages interactive.

Why JavaScript?

  • Adds functionality to webpages
  • Handles user events
  • Manipulates HTML and CSS
  • Creates dynamic websites

Ways to Add JavaScript

  1. Inline JavaScript
  2. Internal JavaScript
  3. External JavaScript
