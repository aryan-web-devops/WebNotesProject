Introduction

HTML stands for Hyper Text Markup Language. It is used to create websites.

Why HTML?

  • Defines structure of webpage
  • Uses tags
  • Creates layout

HTML Document Structure

<html>
   <head>
      <title>Page</title>
   </head>

   <body>
      Content
   </body>
</html>

HTML Tags

Tag Purpose
<p> Paragraph
<img> Image
<a> Link
<br> Line Break
